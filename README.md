# UK_RL

## The First Industrial Revolution in the UK — Speech Materials

This repository contains a complete set of materials for a 5–7 minute English speech on **"The First Industrial Revolution in the UK"**, targeted at university students with basic history knowledge.

### Contents

| File | Description |
|---|---|
| [`presentation/outline.md`](presentation/outline.md) | 5-chapter structured outline with time allocations, transition sentences, and visual aid suggestions |
| [`presentation/speech_script.md`](presentation/speech_script.md) | Full speech script (~800 words) ready for delivery |
| [`presentation/slides_guide.md`](presentation/slides_guide.md) | Slide-by-slide content guide for PowerPoint / Google Slides / Keynote |

### Speech Structure (5 Chapters)

1. **Introduction** — Attention-grabbing opening and roadmap (~1 min)
2. **Background & Causes** — Why the revolution started in Britain (~1 min)
3. **Key Inventions & Breakthroughs** — Technological chain reaction (~1.5 min)
4. **Social & Economic Impacts** — Urbanisation, factory system, and human cost (~2 min)
5. **Legacy & Conclusion** — Modern relevance and closing statement (~1 min)
